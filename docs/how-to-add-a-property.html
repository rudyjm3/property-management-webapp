How to Add a Property

Follow the steps below to add a new property to your portfolio. The entire flow takes less than a minute.

1

Open the Properties page

Click Properties in the left sidebar. The icon looks like a building. The link is highlighted in blue when you're on the Properties page.

The Properties link turns blue and shows a blue background when active. If you don't see + Add Property, your account may have a Maintenance-only role which does not have permission to create properties.
2

Click "+ Add Property"

Click the blue + Add Property button in the top-right corner of the page header. A dialog box will appear over the page.

Field What to enter Required
Property Name A recognizable name for the building or complex (e.g. Sunset Gardens) Yes
Address Street number and street name (e.g. 1200 Sunset Blvd) Yes
City City the property is located in Yes
State Two-letter state abbreviation (e.g. TX). The field caps at 2 characters. Yes
Zip Code 5-digit ZIP code Yes
Type Choose one: Multifamily, Single Family, Commercial, or Mixed Use. Defaults to Multifamily. Yes
The State field is limited to 2 characters. Typing more will not register additional keystrokes — enter only the two-letter abbreviation (e.g. CA, NY, TX).
4

Click "Create Property" to save

Once all fields are filled, click the blue Create Property button. The dialog closes and your new property appears as a card in the grid. To dismiss without saving, click Cancel or click anywhere outside the dialog.
